# Shawn's Portfolio

A dark-themed personal portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks.

## Live Site
[shawnnn25.github.io](https://shawnnn25.github.io)

## Features
- Typing effect in hero section
- Scroll reveal animations
- Responsive navigation with sticky header
- Projects, Skills, About, and Contact sections

## Tech Used
- HTML5
- CSS3 (custom properties, keyframe animations, IntersectionObserver)
- Vanilla JavaScript

## Projects Showcased
- **FlatMate** — Society maintenance management app built with React, Firebase, Razorpay, and Express.js

## Run Locally
No build steps needed. Just clone and open `portfolio.html` in a browser or use Live Server in VS Code.

```bash
git clone https://github.com/shawnnn25/shawnnn25.github.io.git
```
